Comment utiliser votre Backdrop ?

Intégration OBS Studio

  1. Cliquez sur l'URL de copie pour la copier dans votre presse-papier.
  2. Dans OBS Studio, ajoutez une nouvelle source de type Navigateur (Browser Source).
  3. Collez l'URL copiée.
  4. Réglez la résolution sur 1920x1080 (ou votre résolution d'écran), puis validez.

Zoom / Teams / Meet

  1. Réglez les couleurs et le texte selon vos préférences dans le configurateur.
  2. Sélectionnez le format désiré (16:9, Carré ou Bannière).
  3. Cliquez sur Exporter l'Image pour télécharger le fichier PNG.
  4. Dans votre logiciel de visioconférence, ajoutez l'image téléchargée en tant que fond d'écran virtuel.
