Intégration OBS Studio OBS Studio Integration

  1. Cliquez sur l'URL de copie pour la copier dans votre presse-papier. Click on the copy URL to copy it to your clipboard.
  2. Dans OBS Studio, ajoutez une nouvelle source de type Navigateur (Browser Source). In OBS Studio, add a new Source of type Browser (Browser Source).
  3. Collez l'URL copiée. Paste the copied URL.
  4. Réglez la résolution sur 1920x1080 (ou votre résolution d'écran), puis validez. Set the resolution to 1920x1080 (or your screen resolution), then validate.

Zoom / Teams / Meet Zoom / Teams / Meet

  1. Réglez les couleurs et le texte selon vos préférences dans le configurateur. Adjust colors and text to your preferences in the configurator.
  2. Sélectionnez le format désiré (16:9, Carré ou Bannière). Select the desired format (16:9, Square or Banner).
  3. Cliquez sur Exporter l'Image pour télécharger le fichier PNG. Click on Export Image to download the PNG file.
  4. Dans votre logiciel de visioconférence, ajoutez l'image téléchargée en tant que fond d'écran virtuel. In your video conference software, add the downloaded image as a virtual background.
