# 🎥 Backdrop Studio

Virtual meeting backdrop overlay and configurator.

---

## 🚀 Initial Release

Welcome to the initial release of **Backdrop Studio**.

### Features
- Base environment configuration and templates.
- Parameterized production orchestration rules.
- Local Docker developer environments.

### Local Setup
1. Execute `make configure` to setup local packages and configurations.
2. Run `make up` to start the local developer containers.
